What is FIJI?

[Go Back to Table of Contents]

FIJI is a forth-like interpretive procedural language which is in reality a Java interpreter. You can load and excercise Java classes interactively in FIJI, or you can compile FIJI programs to do so.

FIJI is an open source project licensed under the GNU Public License.

Where do I get FIJI?

[Go Back to Table of Contents]

The latest FIJI release is linked from the SoftWoehr Home Page.

Where is the Source Code?

[Go Back to Table of Contents]

A brief note occasioned by this one question that has been asked over and over again!

The soure code is packed into the .jar file with classes!

Please use the Java jar tool, e.g.

jar xf fiji.jar
to unpack the source code if you wish to browse or recompile SoftWoehr Java class libraries.
